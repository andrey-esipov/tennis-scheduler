A beautiful, mobile-first tennis scheduling app built for two players who want a fair commute, great courts, and an easy weekly routine.

![Screenshot](assets/screenshot.png)

## Features
- Smart scheduling that balances commute fairness + court quality
- Real courts across Franklin, Murfreesboro, Smyrna/La Vergne
- Overlapping availability heatmap (D3.js)
- Commute comparison chart (D3.js)
- Weekly calendar with drag-and-drop rescheduling
- Session history, stats, and achievements
- Weather hints for outdoor courts
- All data saved in LocalStorage

## How to Use
1. Open `index.html` in your browser
2. Set each player's availability and court preferences
3. Click **Find best sessions**
4. Add a suggested session to your weekly calendar
5. Track history + stats
